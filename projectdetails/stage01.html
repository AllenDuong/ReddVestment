ReddVestment - Stage 01: Project Details

ER Diagram

Description:

Our project is a web app where users view new Reddit posts and “buy” posts using an artificial point system. Users win points if their post gains karma over the course of N hours, and lose points if it does not gain karma.

Usefulness:

Our application is a game, so it is used for entertainment and to understand what aspects go towards a successful Reddit post. Additionally, it will allow a user to customize their own post, and we would provide suggestions on how they could improve their post.

Realness:

All of our data will be coming from Reddit. We will create a script that parses through the top subreddits and tracks karma, user, title keywords, post-type, outgoing domain and time posted to understand how these different components make a post popular, or unpopular. This data will be used to analyze the average growth of a post given a certain subreddit. We will then use TensorFlow to find trends in the data to predict how successful a post will be.

Functionality:

  1. Basic Functions:
    1. Scrape json metadata from Reddit posts and add the data into our database in a “Reddit” table. A user’s investments are added into a separate “Investment” table.
    2. Reddit data can be deleted manually, and after N weeks the old metadata will be deleted from the database. Investment data for a post is deleted when the user decides to “sell” his or her investment in that post.
    3. Reddit data will be updated such that whenever a new amount of points is recorded on a post, some “karma” list within the “Investment” table is updated to add this new point.
  2. Advanced Functions:
    1. Our first advanced function will be an algorithm to predict the growth of a post based on its title, subreddit, time of post, post-type, outgoing domain, and user. This algorithm will be implemented with a recursive neural network via TensorFlow in order to incorporate karma trends.
    2. Our second advanced function will be a post suggestor. Based on the time of day and subreddit, this function will suggest keywords to include in the title to maximize upvotes. Additionally, we will allow the user to post to reddit through the app with the use of Reddit's API.
