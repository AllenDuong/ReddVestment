ReddVestment - Stage 02: Development Plan

Relational Schema

  • Database Schema:
    • ReddVestment(Posts, Players, Investments) = {Players, Posts → Investments}
  • Table Schema:
    • Posts(subreddit, created, id, author, title, url, score, is_self, gilded) = {id → title}
    • Players(id, name, current_points, number_of_investments) = {id → name}
    • Investments(player_id, post_id, points_invested, current_value, time_of_investment, predicted_value)

Database and Software Platforms:

  • AngularJS
  • Apache Server
  • CSS3 + HTML5 + PHP
  • GitHub
  • MySQL
  • Python
  • TensorFlow ML

Data Acquisition and Source:

  • The source of our data will be directly from Reddit’s servers via parsing JSON formatted data from their API
    • JSON data can be accessed by requesting the following: https://reddit.com/r/[yoursubreddithere].json
  • The top N subreddits, where N is a steadily increasing integer, will be scraped periodically
